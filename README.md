# 🖼 Image Morphing Tool

A simple web-based image morphing application built using Flask and OpenCV.

## 🚀 Features
- Upload two images
- Adjust morphing level (0 to 1)
- Generate blended image

## 🛠 Technologies Used
- Python
- Flask
- OpenCV
- HTML, CSS

## ▶ How to Run

1. Clone repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python app.py
4. Open browser:
   http://127.0.0.1:5000
